# Phase 7 验收清单（前台 + 后台 + 非功能）

> 说明：本清单基于 `docs/requirements.md`、`docs/frontend-implementation-plan.md`、`docs/implementation-plan.md` 与 `docs/phase7-alignment.md` 汇总。
> 勾选说明：请在验收时勾选“通过/不通过”。

## 一、前台功能验收

### 1. 用户注册与登录
- [x] 通过：注册校验（用户名长度、手机号格式）
- [x] 通过：登录成功返回 token，可访问需登录页面
- [x] 通过：token 失效返回 401 并跳转登录

### 2. 用户信息与地址
- [ ] 待确认：用户信息查询/更新（不含密码）
- [ ] 待确认：地址新增/编辑/删除
- [ ] 待确认：首地址自动默认、设默认互斥
- [ ] 待确认：待发货订单地址不可删除

### 3. 图书浏览与检索
- [ ] 待确认：图书列表仅显示上架商品
- [ ] 待确认：支持分类筛选 + 关键词搜索
- [ ] 待确认：分页参数生效（默认/最大值）
- [ ] 待确认：图书详情展示完整信息

### 4. 购物车
- [ ] 待确认：添加校验上架与库存
- [ ] 待确认：同一商品合并数量
- [ ] 待确认：修改数量校验（>0 且 <= 库存）
- [ ] 待确认：购物车列表显示价格/库存

### 5. 订单（前台）
- [ ] 待确认：创建订单（校验地址/库存/购物车）
- [ ] 待确认：支付成功状态流转到待发货
- [ ] 待确认：取消未支付订单
- [ ] 待确认：订单列表与详情仅返回本人数据

### 6. 前台轮播图
- [ ] 待确认：仅返回启用状态轮播图
- [ ] 待确认：最多返回 5 张，排序正确

## 二、后台功能验收

### 1. 管理员认证
- [ ] 待确认：管理员登录角色校验
- [ ] 待确认：后台接口需管理员权限

### 2. 图书与分类管理
- [ ] 待确认：图书/分类 CRUD
- [ ] 待确认：图书上下架、分类启用/禁用
- [ ] 待确认：图书软删除（status = -1）

### 3. 订单管理
- [ ] 待确认：订单列表支持筛选（状态/订单号/用户ID）
- [ ] 待确认：订单详情包含用户信息与明细
- [ ] 待确认：发货仅允许待发货状态

### 4. 用户管理
- [ ] 待确认：用户列表/详情
- [ ] 待确认：用户启用/禁用
- [ ] 待确认：禁用用户无法登录/访问

### 5. 轮播图管理
- [ ] 待确认：轮播图新增/编辑/删除
- [ ] 待确认：排序更新生效

## 三、非功能验收

- [ ] 待确认：统一响应格式（code/message/data/timestamp）
- [ ] 待确认：参数校验失败返回 400
- [ ] 待确认：权限校验返回 401/403
- [ ] 待确认：分页默认/最大值符合规范
- [ ] 待确认：日志不包含敏感信息
