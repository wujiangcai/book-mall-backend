# Phase 7 演示流程走查

> 目标：覆盖前台完整业务链路 + 后台管理流程，确认状态流转正确。

## 一、前台演示流程

1) 注册与登录
- 注册新用户 → 登录 → 进入首页

2) 浏览与搜索
- 进入图书列表 → 分类筛选/关键词搜索 → 进入图书详情

3) 购物车与下单
- 图书详情加入购物车 → 购物车调整数量 → 选择地址 → 创建订单

4) 支付与订单
- 支付订单 → 订单列表查看状态 → 进入订单详情

## 二、后台演示流程

1) 管理员登录
- 管理员账号登录后台

2) 订单管理
- 订单列表筛选（状态/订单号/用户ID）
- 进入订单详情 → 对待发货订单执行发货

3) 用户管理
- 用户列表 → 查看详情 → 启用/禁用

4) 轮播图管理
- 新增轮播图 → 编辑 → 更新排序 → 删除

## 三、前后台联动验证

- 前台支付后订单状态为“待发货”
- 后台发货后前台订单状态变更为“已发货”
